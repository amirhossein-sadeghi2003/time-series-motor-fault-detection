# Dataset Notes

This project uses a subset of the Case Western Reserve University Bearing Fault Dataset for motor bearing fault diagnosis.

The original raw dataset files are MATLAB `.mat` files and are stored locally under:

```text
data/raw/cwru/
```

Raw `.mat` files are not tracked in this repository because they are external dataset files.

---

## Dataset Source

Dataset:

```text
Case Western Reserve University Bearing Data Center
```

Official download page:

```text
https://engineering.case.edu/bearingdatacenter/download-data-file
```

## Processed Dataset

The processed feature dataset is tracked in this repository:

```text
data/processed/cwru_bearing_features.csv
```

It is generated by:

```bash
python src/load_cwru_data.py
```

The script extracts the drive-end vibration signal ending with:

```text
_DE_time
```

Then it applies fixed-size windowing and computes statistical features for each window.

---

## Reproducibility

To reproduce the processed dataset:

1. Download the raw `.mat` files listed above from the official CWRU Bearing Data Center.
2. Place them in the local folder structure shown above.
3. Run:

```bash
python src/load_cwru_data.py
```

This regenerates:

```text
data/processed/cwru_bearing_features.csv
```
